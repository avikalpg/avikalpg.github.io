• OpenSource: Free-time (date: 29 May 2022)

    I have started a free open-source project called Free-Time on Github. It is currently hosted on myfreetimeinaweek.in. The goal of this project is to create a suite of productivity tools which helps individuals perceive and spend their most valuable asset, their time, in a manner that they feel happier and more satisfied with. The project is built in React Native (using Expo). Even though it is only a website at the moment, this tool has the capacity to become a mobile application if the users want to use it frequently.
